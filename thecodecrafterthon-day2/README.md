# BASE CONVERTER CLI (GO)

This is a command line tool written in Go that converts numbers between Decimal, Binary, and Hexadecimal.

## Its Features
. Converts binary - decimal
. Convert hex - decimal
. Convert decimal - binary and hex
. Handles invalid inputs
. Supports negative decimal numbers
. Runs until you type 'quit'

## How to Run
. When you navigat to the project folder "thecodecrafterthon-day2
. Run the program:
go run main.go

## USER USAGE
Command format:
Convert <value> <base>
Supported bases:
bin - binary
hex - hexadecimal
dec - decimal

## Example
> convert 1E hex 
> Output: Decimal: 30 
> convert 10 bin 
> Output: Decimal: 2 
> convert 255 dec
> Output:
Binary: 11111111  
Hex: FF

## Exit
To stop the program:
quit
