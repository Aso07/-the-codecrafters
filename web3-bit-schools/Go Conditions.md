Go Conditions are used to make decisions in a program by executing code only if certain criteria are met.

Go mainly uses if, else if, and else statements for conditional logic.

if checks a condition

else if checks another condition if the first is false

else runs when all previous conditions are false

Conditions must return boolean values (true or false)

## Go if statement

An if statement checks a condition. If the condition is true, the code inside the block runs. However, if it is false, the code is skipped.

## Go if else Statement
The else statement provides an alternative block of code if the if condition does not pass.
else cannot stand alone, it must come after an if.
The { must stay on the same line as else.

## The Nested if Statement
This is used when you want to check a second condition only if the first condition is true. It is in this manner, first check -> if true, then check again -> if true.
