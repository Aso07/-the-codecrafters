# What is a CLI Calculator

This is a command-line calculator built in Go. It supports addition, subtraction, multiplication, and division with proper validation.

## How to Run it:
You can Run this calculator by Opening your terminal and navigate to the project folder where main.go is located.

Run the program using the Go command: "go run main.go

You can Use the calculator by typing commands in this format:
add 5 3
sub 10 4
mul 2 7
div 10 2

You can also get help at any time:
By typing 'help'

One can also exit the calculator at anytime by typing:
'quit'
