# Introduction 
Go, which is also called Golang is a popular programming language, it is used to create fast and efficient computer programs. it was developed at Google by Robert Griesemer, Rob Pike, and Ken Thompson in 2007, but made open source in 2009. 
It is topically a fast, statically typed, compiled language known for its simplicity and efficiency. 
## Its Uses:
Go is used for Web development(server-side) 
It is used for developing network-based programs 
It is used in developing cross-platform enterprise applications 
It is used for Cloud-native development 
## Why use Go: 
It is easy and fun to learn it has a fast run time and compiler time 
It has memory management 
It works in different platforms(e.g Mac, Windows, Linux, e.t.c.)
