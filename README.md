# CODECRAFTERS HACKATHON PROJECT 

This repository contains all my projects completed during the CodeCrafters Hackathon drills using Go (Golang).

The goal of this hackathon was to build practical CLI tools and understand core programming concepts in Go.


## Projects Included

### 1. CLI Calculator

A command-line calculator that performs basic operations.

** Features:

* add, sub, mul, div
* Handles invalid input
* Prevents division by zero
* Supports negative numbers



### 2. String Transformer

A tool for manipulating strings.

** Features:

* Change case (uppercase/lowercase)
* Clean and trim white spaces
* Modify and transform text
* Practice string operations in Go



### 3. Base Converter

A CLI tool that converts numbers between bases.

** Supports:

* Binary → Decimal
* Hex → Decimal
* Decimal → Binary & Hex

** Features:

* Input validation
* Handles invalid values (e.g. 1G, 10201)
* Hex output in uppercase



### 4. W3Schools Go Review

Markdown notes covering Go basics from W3Schools.

** Topics:

* Introduction to Go
* Variables
* Data Types
* Arrays, Slices
* Operators, Conditions



### 5. File Pipeline

A project demonstrating file processing and data flow.

**Features:

* Reads and processes files
* Shows understanding of file handling in Go



### 6. Command & Control CLI

A CLI-based control system for executing commands.

**Features:

* Accepts user commands
* Processes inputs dynamically
* Demonstrates CLI design


## What I Learned

* How to build CLI applications in Go
* Input validation and error handling
* Working with strings and numbers
* File handling in Go
* Structuring programs properly



## How to Run

1. Navigate to any project folder:

```bash
cd thecodecrafterthon-day1
```

2. Run the program:

```bash
go run main.go
```
