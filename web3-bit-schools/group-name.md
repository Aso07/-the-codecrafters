## What is a Goroutine?

A 'goroutine' is a function that runs independently from other functions.

It is started using the keyword `go`.

### Example:

```go
package main

import "fmt"

func sayHello() {
    fmt.Println("Hello from Goroutine!")
}

func main() {
    go sayHello()
}
```

## Key Features of Goroutines

. Lightweight (uses less memory)
. Fast execution
. Supports concurrency
. Easy to use
