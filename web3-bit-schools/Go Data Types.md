Go Data Types define the kind of value a variable can hold, such as numbers, text, or true/false values.

## Common Go data types:

* Integer Data Type – integer numbers (e.g., 10, -5), Integer Data Type in Go is used to store whole numbers (no decimals).
Go provides several integer types with different sizes and ranges:
int – default integer type (size depends on the system)
int8 – 8-bit integer (-128 to 127)
int16 – 16-bit integer (-32,768 to 32,767)
int32 – 32-bit integer (-2,147,483,648 to 2,147,483,647)
int64 – 64-bit integer (-9,223,372,036,854,775,808 to 9,223,372,036,854,775,807)
uint, uint8, uint16, uint32, uint64 – unsigned integers (only positive numbers, including 0)


* Go Float Data Types – decimal numbers (e.g., 3.14), The float data types are used to store positive and negative numbers with a decimal point, like 35.3, -2.34, or 3597.34987.


* String Data Type – text (e.g., "Hello"), String Data Type in Go is used to store text, which is a sequence of characters.
Strings are declared using the string type and are enclosed in double quotes ("").


* Boolean Data Type – boolean values (true or false). Bool is used for conditions, flags, or any situation with true/false values.
