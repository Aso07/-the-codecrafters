# What is a Function
A function is just a way to group some code and give it a name.
You give it something (input)
It does some work
It may give you something back (output)

## Why Function is Important
We need functions because they:

Reduce repetition
Make code readable
Organize programs
Make fixing bugs easier

Hence, a function take input, do something, optionally return output, and can be reused anytime.
